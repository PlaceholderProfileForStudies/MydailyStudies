# Loopings
**Para fazer o loop while:**  
    **Comando:**  
```java
    public static void main(String[] args) {
            int contador = 0;
            while(contador <=10) {
                System.out.printf("Estou no valor %d!\n",contador);
                contador++;
            }
        }
```
**Para fazer o loop for:**  
    **Comando:**  
```java
    for(int i = 0;i<=10;i++) {
			System.out.printf("Estou no valor %d!\n",i);
		}
	}
```
**Para fazer o loop Do while:**
