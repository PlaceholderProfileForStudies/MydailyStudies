# Switch
1. **Como fazer a condição `Switch`:**  
    **Comando:**  
    ```java
        public static void main(String[] args){
            int vida = 100;
            switch(vida) {
                case 50:
                    System.out.println("A sua voda está na metade!");
                    break;
                case 25:
                    System.out.println("A sua vida está baixa!");
                    break;
                default:
                    System.out.println("Sua visa está cheia!");
                    break;
		}
	}	
    ```
