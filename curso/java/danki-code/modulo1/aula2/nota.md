# Variáveis
1. **Para criar uma variável:**  
    **Comando:**  
    ```java
        public static void main(String[] args) {
		int idade = 18;
	}
    ```
2. **Para usar variável junto com texto:**  
    **Comando:**  
    ```java
        String mensagem = String.format("Você tem %d anos!",idade);
		System.out.println(mensagem);
    ```
